# Анализ продаж сети Walmart — Python

Дипломный проект курса «Дата-аналитик» (Tehnikum School).
Исследование продаж 45 магазинов Walmart за 2010–2012 годы.

## Данные
- 6 435 строк, 45 магазинов, период 2010–2012
- Показатели: еженедельные продажи, уровень безработицы, CPI, праздники

## Инструменты
- Python: Pandas, NumPy, Matplotlib
- Jupyter Notebook

## Этапы работы
1. **Очистка данных** — обработка пропусков, приведение типов
2. **EDA** — сравнение магазинов по среднему недельному обороту
3. **Корреляционный анализ** — зависимость продаж от безработицы и CPI

## Ключевые выводы
- Лидеры продаж: свыше **$2 млн/неделю** (на 100–115% выше среднего)
- Отстающие: менее **$310 тыс./неделю** — в 7 раз ниже лидеров
- Выявлено **3 магазина** с высокой чувствительностью к экономике (r = 0,7–0,9)
- Группа **устойчивых магазинов** с r < 0,1 — не зависят от макроэкономики
